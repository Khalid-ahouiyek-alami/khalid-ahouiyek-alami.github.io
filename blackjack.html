BlackJack Game

Een Python Command Line Kaartspel

Python
CLI

Project Overzicht

Een command-line implementatie van het klassieke BlackJack kaartspel, ontwikkeld in Python. Het spel biedt een interactieve spelervaring waarbij spelers kunnen kiezen tussen 'hit' of 'stand', terwijl ze proberen de dealer te verslaan zonder over de 21 te gaan.

Spelkenmerken

Gameplay Mechanismen

  • Hit of Stand keuzes
  • Automatische score berekening
  • Dealer AI logica
  • Aas waarde aanpassing (1 of 11)

Technische Features

  • Random kaart generatie
  • Score tracking systeem
  • Input validatie
  • Duidelijke gebruikersinterface

def calculate_score(cards):
    score = sum(cards)
    if score == 21 and len(cards) == 2:
        return 0  # BlackJack
    if 11 in cards and score > 21:
        cards.remove(11)
        cards.append(1)
    return score

def deal_card():
    cards = [11, 2, 3, 4, 5, 6, 7, 8, 9, 10, 10, 10, 10]
    return random.choice(cards)

def play_game():
    user_cards = []
    computer_cards = []
    
    for _ in range(2):
        user_cards.append(deal_card())
        computer_cards.append(deal_card())
    
    game_over = False
    while not game_over:
        user_score = calculate_score(user_cards)
        computer_score = calculate_score(computer_cards)
        
        print(f"Jouw kaarten: {user_cards}, score: {user_score}")
        print(f"Computer's eerste kaart: {computer_cards[0]}")
        
        if user_score == 0 or computer_score == 0 or user_score > 21:
            game_over = True
        else:
            should_deal = input("Type 'y' voor nog een kaart, type 'n' om te passen: ")
            if should_deal.lower() == 'y':
                user_cards.append(deal_card())
            else:
                game_over = True

Deze functie toont hoe het spel de score berekent en omgaat met de speciale Aas-kaart regels, waarbij een Aas zowel 1 als 11 kan zijn afhankelijk van de situatie.
